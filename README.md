# css-advanced-selectors

This project is a single-page webpage created as part of a class assignment focused on practicing **advanced CSS selectors** instead of using classes or IDs.

The page demonstrates how CSS selectors such as **nth-child, first-child, pseudo-classes, and structural selectors** can be used to style elements based on their position, type, and interaction state.

---

## Page Overview
- **Home Page**:  
  Includes two lists (unordered and ordered), three styled content boxes containing paragraph text, and an interactive button that changes appearance based on user actions such as hover and click.

## Assignment Objectives
This project demonstrates:
- Styling list items using `:nth-child()` selectors
- Styling specific elements without using classes or IDs  
- Using `:first-child` to style the first paragraph in each container  
- Targeting specific elements using `:nth-of-type()`  
- Creating interactive button effects using `:hover` and `:active`
